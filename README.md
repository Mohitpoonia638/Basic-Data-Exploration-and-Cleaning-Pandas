# Basic Data Exploration and Cleaning using Pandas

## Objective

Learn Python basics and perform basic data exploration and cleaning using Pandas.

## Dataset

Combined_dataset.csv

## Tasks Performed

1. Loaded the dataset into a Pandas DataFrame.
2. Explored the dataset using:
   - head()
   - tail()
   - shape
   - columns
   - info()
3. Identified missing values.
4. Handled missing values using fillna().
5. Removed duplicate records.
6. Performed column selection.
7. Filtered rows based on conditions.
8. Created a derived column named `discount_amount`.
9. Exported the cleaned dataset as `cleaned_shopping.csv`.
